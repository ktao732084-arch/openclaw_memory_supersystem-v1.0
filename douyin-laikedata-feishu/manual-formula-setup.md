## 第二步：配置公式字段（需要手动操作）

### 操作步骤

1. **打开飞书多维表格**
   - 访问：https://ocnbk46uzxq8.feishu.cn/base/FEiCbGEDHarzyUsPG8QcoLxwn7d
   - 打开"投放数据"表格

2. **添加公式字段1：客资数量**
   - 点击表格右侧的"+"添加字段
   - 字段名称：`客资数量`
   - 字段类型：选择"公式"
   - 公式内容：
     ```
     COUNTIFS(客资数据.单元ID前15位, LEFT(单元ID, 15), 客资数据.日期, 时间)
     ```
   - 点击"确定"

3. **添加公式字段2：实际获客成本**
   - 点击表格右侧的"+"添加字段
   - 字段名称：`实际获客成本`
   - 字段类型：选择"公式"
   - 公式内容：
     ```
     IF(客资数量 > 0, 消耗(元) / 客资数量, 0)
     ```
   - 点击"确定"

4. **添加公式字段3：客资转化率(%)**
   - 点击表格右侧的"+"添加字段
   - 字段名称：`客资转化率(%)`
   - 字段类型：选择"公式"
   - 公式内容：
     ```
     IF(客资数量 > 0, (转化数 / 客资数量) * 100, 0)
     ```
   - 点击"确定"

## 注意事项

### 字段名称必须完全匹配
- `客资数据` - 客资数据表的名称
- `单元ID前15位` - 客资数据表中的字段
- `单元ID` - 投放数据表中的字段
- `时间` - 投放数据表中的日期字段
- `消耗(元)` - 投放数据表中的字段
- `转化数` - 投放数据表中的字段

### 如果字段名称不匹配
请根据实际字段名称调整公式。
