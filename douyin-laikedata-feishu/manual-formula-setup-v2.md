### 公式1：客资数量

**字段名称**：`客资数量`  
**字段类型**：公式

**公式内容**：
```
[客资数据].FILTER(
  AND(
    CurrentValue.[单元ID前15位] = LEFT([单元ID], 15),
    CurrentValue.[日期] = [时间]
  )
).COUNTA()
```

**说明**：
- 从客资数据表中筛选出：单元ID前15位匹配 且 日期匹配的记录
- 统计筛选后的记录数量
- `CurrentValue` 表示客资数据表中的当前记录
- `LEFT([单元ID], 15)` 取投放数据表中单元ID的前15位

---

### 公式2：实际获客成本

**字段名称**：`实际获客成本`  
**字段类型**：公式

**公式内容**：
```
IF([客资数量] > 0, [消耗(元)] / [客资数量], 0)
```

**说明**：
- 如果客资数量大于0，则计算：消耗 ÷ 客资数量
- 否则返回0

---

### 公式3：客资转化率(%)

**字段名称**：`客资转化率(%)`  
**字段类型**：公式

**公式内容**：
```
IF([客资数量] > 0, ([转化数] / [客资数量]) * 100, 0)
```

**说明**：
- 如果客资数量大于0，则计算：(转化数 ÷ 客资数量) × 100
- 否则返回0

## 注意事项

### 1. 字段名称必须完全匹配
- 包括括号、空格、大小写
- 如果字段名称不同，需要修改公式中的字段名

### 2. 表格名称必须正确
- 客资数据表的名称必须是"客资数据"
- 如果名称不同，需要修改公式中的表格名

### 3. 公式语法说明
- `[表格名]` - 引用其他表格
- `[字段名]` - 引用当前表格的字段
- `CurrentValue.[字段名]` - 在FILTER中引用被筛选表格的字段
- `LEFT(文本, 长度)` - 提取文本左侧指定长度的字符
- `COUNTA()` - 统计非空值的数量
