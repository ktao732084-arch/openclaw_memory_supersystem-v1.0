## 公式1：客资数量

**字段名称**：`客资数量`  
**字段类型**：公式

**公式内容**：
```
[Sheet2].[手机号].FILTER(
  AND(
    LEFT([Sheet2].[单元ID], 15) = LEFT([单元ID], 15),
    TEXT([Sheet2].[线索创建时间], "YYYY-MM-DD") = TEXT([时间], "YYYY-MM-DD")
  )
).COUNTA()
```

**说明**：
- 从 Sheet2 的"手机号"字段中筛选
- 条件1：单元ID前15位匹配
- 条件2：日期匹配（将线索创建时间转换为日期格式）
- 统计筛选后的记录数

---

## 公式2：实际获客成本

**字段名称**：`实际获客成本`  
**字段类型**：公式

**公式内容**：
```
IF([客资数量] > 0, [消耗(元)] / [客资数量], 0)
```

---

## 公式3：客资转化率(%)

**字段名称**：`客资转化率(%)`  
**字段类型**：公式

**公式内容**：
```
IF([客资数量] > 0, ([转化数] / [客资数量]) * 100, 0)
```

## 注意事项

### 1. 日期格式转换

Sheet2 中的"线索创建时间"格式是："2026-01-21 14:57:05"  
投放数据表中的"时间"格式是："2026-02-01"

使用 `TEXT()` 函数将两者都转换为 "YYYY-MM-DD" 格式进行比较。

### 2. 单元ID匹配

使用 `LEFT()` 函数取前15位进行匹配，避免科学计数法问题。

## 测试验证

配置完成后：

1. 查看投放数据表中的任意一条记录
2. 检查"客资数量"字段是否显示数字
3. 如果显示0，可能是因为：
   - 该单元ID在Sheet2中没有对应的客资
   - 日期不匹配
